# Python Log Analyzer (Bulk Analysis)

This project focuses on bulk analysis of authentication logs using Python.
It does NOT perform threat detection or alerting.

## What This Tool Does
- Reads authentication logs
- Counts total log entries
- Separates failed and successful logins
- Summarizes IP activity

## Use Case
- Log analysis
- Data preparation for SOC detection rules
- Understanding authentication behavior patterns
